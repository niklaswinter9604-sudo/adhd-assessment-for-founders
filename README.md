# adhd-assessment-for-founders
Am Assessment for founders with ADHD. The result provides a System Prompt for an antagonist Copilot that supports your work based on your neurodivergence.



ADDITIONAL DISCLAIMER:

This software is a self-assessment tool for educational and self-reflection 
purposes only. It is NOT a medical diagnostic tool. The software does not 
store, transmit, or process any personal data beyond the user's local browser 
session. All data is permanently deleted when the browser tab is closed.
